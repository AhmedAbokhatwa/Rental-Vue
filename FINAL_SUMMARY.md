## 🛠️ التقنيات المستخدمة

### Frontend

- **Node.js** - خادم التطوير (اختياري)

## 🔧 الإعداد والتشغيل

### المتطلبات
- Node.js 16+
- npm أو yarn
- خادم Frappe/ERPNext
- متصفح حديث

### خطوات التشغيل
1. تثبيت التبعيات: `npm install`
2. إنشاء ملف `.env` مع الإعدادات
3. تشغيل التطبيق: `npm run dev`
4. فتح المتصفح على `http://localhost:5173`
